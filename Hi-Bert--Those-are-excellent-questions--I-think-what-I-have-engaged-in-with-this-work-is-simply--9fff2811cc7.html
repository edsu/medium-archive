Social Media Authors, Authority and the Archive


Hi Bert. Those are excellent questions. I think what I have engaged in with this work is simply collection building. I agree that significant work needs to done to think of it as an archive, and the majority of this work necessarily involves a meaningful relationship with the content creators, the authors who have the authority.

It is somewhat problematic because there are so many authors. Perhaps this is heretical thinking, but if you think of the Web as an archive, and authors putting their content out into the space of the Web, then perhaps we can see this act on their part as an expression of their intent? That being said I think we can come up with models and tools that allow archivists to engage with the array of content creators on social media. This is where some really interesting work can be done.

One side note about this application I built to display random #FreddieGray tweets is that if a Twitter user deletes their content it will no longer be displayed. This is a simple side effect of using Twitter’s embed API to display the tweets. I have not saved the images or video, but that is the direction Bergis and I are hoping to explore. Keep the questions coming! It would be interesting to chat about this work sometime.
